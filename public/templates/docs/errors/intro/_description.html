Errors



Conventional HTTP response codes are used to indicate success or failure of an API request. Codes in the 2xx range indicate success, codes in the 4xx range indicate an error that resulted from the provided information (e.g. a required parameter was missing, a charge failed, etc.), and codes in the 5xx range indicate an error with Hello Block's servers.


Response Attributes

details: string
error: string
inspect: string
contribute: string
backtrace: string
