Get Wallet


Gets and returns addresses, transactions, unspents for multiple addresses.


Request Parameters

Param Name Default Value Type Required
addresses: array of strings
repeat for multiple addresses
 REQUIRED
transactions: false boolean OPTIONAL
unspents: false boolean OPTIONAL
limit: 10 integer OPTIONAL
offset: 0 integer OPTIONAL

Response Attributes for Wallet

summary:
balance integer
balance (incl. unconfirmed) for all addresses
confirmedBalance integer
confirmedBalance for all addresses
txsCount integer
txsCount for all addresses
confirmedTxsCount integer
confirmedTxsCount for all addresses
addresses: array of Address Objects for all addresses
transactions: array of Transaction Objects for all addresses
unspents: array of Unspent Objects for all addresses
