Retrieve Unspent Outputs


Gets and returns the unspent outputs for a given address.


Request Parameters

Param Name Default Value Type Required
address: string REQUIRED
limit: 10 integer OPTIONAL
offset: 0 integer OPTIONAL

Response Attributes for Unspents

confirmations: integer
blockHeight: integer
prevTxHash: string
prevTxOutIndex: integer
scriptPubKey: string
type: string
value: integer
hash160: string
address: string
