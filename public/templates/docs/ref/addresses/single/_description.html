Retrieve an Address


Gets and returns data for a single address.


Request Parameters

Param Name Default Value Type Required
address: string REQUIRED
limit: 10 integer OPTIONAL
offset: 0 integer OPTIONAL
transactions: true boolean OPTIONAL
unspents: true boolean OPTIONAL

Response Attributes for Address

balance: integer
includes unconfirmed transactions
confirmedBalance: integer
includes only confirmed transactions
transactionsCount: integer
confirmedTransactionsCount: integer
address: string (base58)
hash160: string
unspents: array of Unspent Objects
transactions: array of Transaction Objects
