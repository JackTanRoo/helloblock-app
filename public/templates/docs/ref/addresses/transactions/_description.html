Retrieve Transactions


Batch gets and returns the transactions for multiple addresses.


Request Parameters

Param Name Default Value Type Required
addresses: array of strings REQUIRED
limit: 10 integer OPTIONAL
offset: 0 integer OPTIONAL

Response Attributes for Transactions

txHash: string
version: integer
size: integer
blockHash: string
blockHeight: integer
blockTime: integer
estimatedTxTime: integer
confirmations: integer
inputsCount: integer
outputsCount: integer
inputs: array of input objects (see below)
inputs: array of output objects (see below)
totalInputsValue: integer
totalOutputsValue: integer
fees: integer
estimatedTxDirection:
string
Related to address/es given
"incoming" || "outgoing"
estimatedTxValue: integer
Related to address/es given
