DEFINITION
  
GET https://api.helloblock.io/v1/addresses/unspents?addresses[]=<address>
EXAMPLE REQUEST
  
$ curl -G https://api.helloblock.io/v1/addresses/unspents \
   -d "addresses[]=mm7skHNwsbMSGxZA7Au4Gp5sRdNkfLQkLZ" \
   -d "addresses[]=miwCPtnuCKxn5JMxXaGZumeHhSkndz6FTP" \
   -d "addresses[]=msv7sCECsEC8vzAoPUkGVypXoxUBagFjYG"
