Retrieve multiple Addresses


Gets and returns data for multiple addresses.


Request Parameters

addresses[]: string [required]

Repeat for multiple addresses, e.g.
"addresses[]=&addresses[]=&addresses[]=c"
txs: boolean [optional; default: true]
wallet: boolean [optional; default: true]
info: boolean [optional; default: true]

Response Attributes

wallet [optional]:

Wallet Object

tx_n integer
total_received_n integer
total_received integer
total_sent_n integer
total_sent integer
balance integer
addresses:

array of Address Objects
txs [optional]:

array of Transaction Objects
info [optional]:

Info Object

tx_n integer
total_received_n integer
total_received integer
total_sent_n integer
total_sent integer
balance integer
